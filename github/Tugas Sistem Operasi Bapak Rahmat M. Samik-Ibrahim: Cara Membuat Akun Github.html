Cara Membuat Akun Github

Berikut langkah - langkah untuk membuat sebuah akun github :
1. silahkan buka github.com pada browser. klik link "Join Github". pastikan anda sudah memiliki akun email sebelumnya.













2. pada halaman yang muncul silahkan masukan username, email serta password yang akan digunakan nantinya ketika akan membuat sebuah repository. setalah data dimasukan silahkan klik tombol "Create an Account".













3. pada bagian chose personal plan silahkan pilih "unlimited public repository for free" untuk akun yang tidak berbayar. kemudian klik tombol "Finish Sign Up".













4. Akun github telah selesai dibuat. silahkan konfirmasi akun yang dibuat melalui link yang akan dikirimkan oleh pihak github.com ke alamat email yang anda daftarkan tadi. setelah di konfirmasi anda sudah dapat membuat repository.
